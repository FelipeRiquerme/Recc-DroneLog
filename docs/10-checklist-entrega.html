Checklist técnico

Projeto abre na IDE e inicia sem erro impeditivo.
Banco PostgreSQL configurado corretamente.
Entidades e relacionamentos fazem sentido para o contexto.
Repositories, services e controllers estão organizados.
CRUDs principais foram testados.
DTOs de entrada e saída foram usados adequadamente.
Validações básicas estão funcionando.
Erros previsíveis possuem tratamento compreensível.
Swagger abre e mostra as rotas.
Filtros implementados foram testados.

Checklist de entrega

  • Repositório no GitHub atualizado.
  • Link correto enviado no AVA.
  • Código sem arquivos desnecessários de build.
  • Commits registrados com mensagens minimamente compreensíveis.
A entrega será analisada pelo funcionamento, pela organização e pela coerência das decisões no código.
