Operações comuns

OperaçãoMétodo HTTPIdeia geral
ListarGETRetorna vários registros, podendo aceitar filtros.
Buscar por idPilotoGETRetorna um registro específico.
CadastrarPOSTCria um novo registro a partir de dados de entrada.
AtualizarPUTAltera um registro existente.
ExcluirDELETERemove ou invalida um registro, conforme a regra adotada.

Pontos de atenção

  • Buscar por idPiloto deve tratar o caso de registro inexistente.
  • Cadastrar e atualizar devem respeitar os DTOs de entrada.
  • A resposta deve usar DTO de saída, evitando exposição desnecessária da entidade.
  • Relacionamentos exigem cuidado: geralmente é necessário buscar a entidade relacionada antes de salvar.
Não basta a rota existir. Ela precisa executar o fluxo completo e devolver uma resposta adequada.
