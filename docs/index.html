DroneLog

Materiais de consulta para recuperação prática em API Spring Boot

Importante: estes materiais servem para consulta geral. A solução deve ser construída analisando o código, os TODOs e a estrutura do projeto.

Contexto da API

A aplicação representa uma base simples para controle de operações com drones em campo. O sistema envolve drones, pilotos e missões de voo.

O projeto possui partes com diferentes níveis de acabamento: alguns arquivos estão mais completos, outros estão incompletos e outros exigem maior autonomia.
