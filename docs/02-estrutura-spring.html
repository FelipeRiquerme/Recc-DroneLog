Camadas esperadas

PacoteResponsabilidade
modelRepresenta as entidades persistidas no banco.
repositoryFornece acesso aos dados usando Spring Data JPA.
dtoDefine dados de entrada e saída da API.
serviceCentraliza regras, buscas, conversões e fluxo principal.
controllerExpõe as rotas HTTP.
exceptionOrganiza erros tratados pela aplicação.
configConfigura recursos auxiliares, como documentação.

Fluxo de uma requisição

Cliente da API
   ↓
Controller
   ↓
Service
   ↓
Repository
   ↓
Banco de dados
Boa prática: evite resolver tudo no controller. Ele deve receber a requisição, encaminhar para o service e devolver a resposta.

Como estudar o código base

Compare arquivos parecidos. Quando uma entidade estiver mais completa, ela pode indicar um padrão geral de organização. Use esse padrão com cuidado, adaptando ao contexto de cada recurso.
